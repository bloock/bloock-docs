# Golang SDK Quickstart

This quickstart shows how to use BLOOCK's Managed API to add live video and
audio to your Android applications.

## Objective

You'll learn how to:

- Install the Bloock SDK
- Initialize the SDK
- Configure a Bloock meeting
- Initialize the Bloock meeting
- And go live with your Bloock meeting
