# Golang SDK

This page has all the information needed to install and setup the BLOOCK's SDK for Golang. If you are looking for documentation related to an specific functionality, please refer to it's specific section.

## Requirements

To run the SDK for Golang you will need:

- Go version >= 1.18

:::tip
Please refer also to [OS support](../overview/os-support.md) for current operating system requirements
:::

## Installation

The Go SDK is available through the go get command, to install it, you can run the following command:

<CodeBlock language="bash">go get github.com/bloock/bloock-sdk-go/v2</CodeBlock>

## Set up

To start working with most of the functionalities of our SDKs, you will need an API Key. Please, follow the instructions of [Create an API Key](../authentication/create-an-api-key.md) to get one.
