### Step 1: Register in the Developer Portal

To get started with Bloock, you must first register an account on the [Developer portal](https://dev.bloock.com).

### Step 2: Create an Organisation

After you have created your account on Bloock, go ahead and create an orgnisation. All meeting, sessions, logs and recordings are created within an organisation.

### Step 3: Get your API Keys

Copy the [API Keys](https://dev.bloock.com/apikeys) for your organization from
the Developer portal. The API Key is unique to your organization and is
required as an authorization header to use Bloock REST APIs.
