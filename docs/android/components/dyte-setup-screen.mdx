# DyteSetupFragment

A screen shown before joining the meeting, where you can edit your display name,
and media settings.

To show DyteSetupFragment do in kotlin/Java as follows:
```kotlin
val dyteSetupFragment = DyteSetupFragment()
dyteSetupFragment.show(
  fragmentManager,
  "SOME_TAG_HERE"
)
```
