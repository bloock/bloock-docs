Bloock's live video platform, built upon [WebRTC](https://bloock.com/blog/webrtc/), enables you to seamlessly incorporate real-time communication features into your web and mobile applications. Using Bloock's REST APIs and client-side SDKs, you can effortlessly acquire, transmit, and play media while efficiently managing network connections within your applications. All Bloock live video calls come with in-built noise cancellation (powered by Krisp).

## Supported platforms

Bloock currently supports the following platforms:

- **Mobile**: Flutter, Android (Java/Kotlin), iOS (Objective-C/Swift), React Native
- **Web**: Javascript Core SDK + UI Kit for React JS, Angular, and Web Components for other frameworks
- **Desktop**: Electron
