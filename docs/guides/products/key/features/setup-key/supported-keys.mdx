# Supported keys

BLOOCK currently supports different key types. You can choose the one it better accomplishes your requirements.

| Type | Key type                     |
| ---- | ---------------------------- |
| EC   | P-256K                       |
| RSA  | 2048-bit, 3072-bit, 4096-bit |
| AES  | 128-bit, 256-bit             |
| BJJ  | BN128                        |
